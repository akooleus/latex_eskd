Эта директория зарезервирована под заполненные примеры.

Базовый сценарий:

1. скопировать `../template_project`;
2. переименовать папку под конкретную работу;
3. при необходимости отредактировать `project_manifest.toml`;
4. заполнить `project_meta.tex`;
5. отредактировать `docs/note/sections/*`, `docs/bom/items.json`,
   `docs/drawing/fragments/*` или положить `docs/drawing/drawing.pdf`;
6. запустить `python3 build_bundle.py`.
